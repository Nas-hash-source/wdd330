Chapter 5: Object

Object Literals

We create an object literal by entering a pair of curly braces. const spiderman = {};. ES6 provided a shorthand of creating object if a property key is the same as a variable name: const ironMan = { name, realName };

We can access the property by using the dot notation (superman.name or with bracket notation supername['name'].

Computed Properties allows an object to have key as a code inside square brackets

We call a method by using the dot notation and add parentheses after the method name.

Check the existence of a property or method: 

                
                    'city' in superman; //using the in operator
                    superman.city !== undefined; //checking if it is undefined
                    superman.hasOwnProperty('city'); //use the hasOwnProperty method

Use for in loop for iterating through all of an object's properties: it allows to find all of the values of the key of the object, however, there may be inherited properties, so, it's better to use Object.keys(object) for (const key of Object.keys(superman))

Object.entries() also used to access the key-value pair

Adding properties: superman.city = 'Metropolis';//add by assigning value
We can change any properties of an object also by assigning to another
Delete a property: delete superman.fly

We can have nested objects in your liking, and we can use object as parameters to functions,

the concept "this" refers to the object properties and method within while used in that same object.

            JSON.parse(batman); //change the JSON format to object
            JSON.stringify(wonderWoman); //change an object to JSON format

                Math.PI // an example of Math property

Chapter 6: The Document Object Model

The DOM provides several methods to access all of the HTML nodes, for example: 

            
                const body = document.body; // get the body
                body.nodeType; // to see the type of the node
                body.nodeName; // to see the name of the element
                const imageArray = Array.from(document.images); // to turn nodeList to array
                const h1 = document.getElementById('title'); // get the element by its id
                const listItems = document.getElementsByTagName('li') // get element by their tag name
                document.querySelector('#bats') //we get the element by using selector like in css
                const wonderWoman = document.querySelector('ul#roster li:last-child');
                wonderWoman.parentNode; // access the parent
                wonderWoman.nextSibling // access the next sibling

Finding the value of a node: 

            
                const textNode = wonderWoman.firstChild;
                textNode.nodeValue;
                wonderWoman.textContent; //find the value of wonderWoman
                wonderWoman.getAttribute('class'); // returns the value of the attribute
                wonderWoman.setAttribute('class', 'villain'); // set the class attribute to villain

The className Property: 

            
                wonderWoman.className; //assign the class attribute directly
                wonderWoman.classList.add('warrior'); // add a class to a specific element
                wonderWoman.classList.toggle('hero'); // will remove the hero class if it exists in the element
                wonderWoman.classList.toggle('sport'); // will add the 'hero' class back

Creating an element: 

            
                const flash = document.createElement('li');
                const flashText = document.createTextNode('Flash'); //create the text node
                flash.appendChild(flashText);
This can be made simpler in this following code: 
            
                const flash = document.createElement('li');
                flash.textContent = 'Flash';

The insertBefore() method appends an element before a specific element

Updating CSS: it should have a style property following by Camel Case CSS Properties example: superman.style.backgroundColor = 'blue';

Chapter 7: Events

It adds interactions between the web page and the users.

            
                document.body.addEventListener("click", doSomething); // when clicked, the doSomething function runs 
                function doSomething(event){
                    console.log(event.target); //event.target refers to the node that fired the event.
                }

Types of Events:
  • click: when a mouse button is clicked.
  • mousedown: when a mouse button is down.
  • mouseup: when a mouse button is up.
  • dblclick: when doubleclicked.
  • mouseover: when hovered
  • mouseout: after hovered, and back again when it moves away
  • keydown: when a key is pressed.
  • keyup: when the button is released.
  • event.key: the key pressed at that moment.
  • removeEventListener() // remove an event

event.preventDefault() is a method of the event object to stop the default behavior happening such as link and submit button.
