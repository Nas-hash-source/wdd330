Transforms

It will enable movement such as skewing, rotation, translation and scaling.

  • Translation: works only on a block display.

    movement through the right, left, bottom and top.

                    -webkit-transform: translate(45px,-45px); /* iOS8, Android 4.4.3, BB10 */
                -ms-transform: translate(45px,-45px); /* IE9 only */
                transform: translate(45px,-45px); // 45px from the left, -45px to the top
                transform: translateX(45px); // only x
                transform: translateY(-30px); // only y
  • Scaling:

    scales an element horizontally and/or vertically.

                    transform: scale(1.5, 0.25);
                transform: scale(2); // if one value is provided,
                // it will be used for both x and y
                // scale(2) double the size of the object
  • Rotation:

    rotate the thing in a x deg

                    transform: rotate(90deg);
  • Skew:

    skew along the x axis and y axes

                    transform: skew(15deg, 4deg);
                transform: skewX(15deg) skewY(4deg);
  • Transform-origin:

    By default, the transform origin is from the center by default.

                    transform-origin: 0 0; // (y, x)

Transitions:

It allows the changing over time. Here are the steps to create a simple transition:

  • declare the original state
  • declare the final state
  • include the transition functions in the original state

  • transition-property: 
                  .ad-ad2 h1 span {
                  -webkit-transition-property: -webkit-transform; // support older browsers
                  transition-property: transform;
                }  // seperated the property you want to transition with comma
  • transition-duration: transition-duration: 0.2s; //200ms 200 ms is considered as the optimum time, however, making slower will make the website seem slow.
  • transition-timing-function:

    • It adds details to the animation: either starting off fast and ending slower, or starting slow and ending faster or some other variations.

                    transition-timing-function: ease; // slow start, fast end
                // ease-in-out, same as ease but a little bit faster in the start
                // linear, constant speed
                // ease-in, slow to start - gains speed - stops abruptly
                // ease-out, full speed - slows progressively
                // precise it with the cubic of cubic-bezier(0, 0, 1, 1)
  • transition-delay property: 
                    -webkit-transition-delay: 50ms;
                transition-delay: 50ms; // delay the transition
  • transition shorthand property: 
                    
                    transition: transform 0.2s ease-out 50ms;
                    transition: transform 0.2s ease-out 50ms, color 0.1s linear 50ms;
                    transition: all 0.2s ease-out 50ms; // all property

Animations

Unlike transitions, controls at each state the animations via keyframes

            // customized a named animation
            @-webkit-keyframes myAnimation {
                /* put animation keyframes here */
               }
            @keyframes myAnimation {
                /* put animation keyframes here */
               }
            @keyframes moveRight {
                from { // 0%
                  transform: translateX(-50%);
                }
                to { // 100%
                  transform: translateX(50%);
                }
            }
            @keyframes appearDisappear {
                0%, 100% { // selector of percentage,
                    // it means that the start and the end will be invisible
                  opacity: 0;
                }
                20%, 80% { // here, it will be visible
                    opacity: 1;
                }
            }
            @keyframes bgMove {
                100% {
                  background-position: 120% 0;
                }
            }

Animation Properties:

  • animation-name: name; attach the keyframe name.
  • animation-duration: length of the time from 0% to 100%.
  • animation-timing-function: like the transition property one.
  • animation-iteration-count: 
        animation-iteration-count: infinite;
    // number of the repetion of the animation
  • animation-direction: set the direction of the iteration 
    
animation-direction: reverse; // 100% to 0%
animation-duration: alternate; // alternate the nomal and reverse starting from 0% to 100% 
animation-duration: alternate-reverse; // alternate the direction starting from 100% to 0%
  • animation-delay: delay before the animation
  • animation-fill-mode:

    it indicates what happens before and after the animation. 

    animation-fill-mode: none; // normally as expected
animation-fill-mode: backwards; // the state will change directly to the first modification
// tonga dia zay fiovana volohany eo no miseho voalohany
animation-fill-mode: forwards; // the final change will be set to the element
animation-fill-mode: both; // backwards, then forwards

  • animation-play-state: running or paused
  • shorthand property: animation: 300ms ease-in 1 alternate 5s backwards appearDisappear running; // note that when it's default, you can omit the property.

Drag and Drop

Make an image draggable, set its attribute to true

Set an event to the image, with "dragstart" handler

    mouse.addEventListener("dragstart", function (event) {
        event.dataTransfer.setData("text/plain", this.id);
    }); // then the dataTransfer set the mime type, and the id of the draggable element
    

    // set the dragover event to the location we are going to drop
    var cat = document.getElementById("cat");
    cat.addEventListener("dragover", function(event) {
        event.preventDefault(); // prevent the default action
    });

    // now an event when it is dropped
    cat.addEventListener("drop", function(event) {
        var mouseID = event.originalEvent.dataTransfer.getData("text/plain");
    } // then get the data we have set.
